# download-for-offline-webmcu-ai
download-for-offline-webmcu-ai


Unlike a live webpage PWA's can work offline but they cannot access files on your computer unless you specifically load them.
When offline you theoretically might not have access to the original [webmcu-ai](https://github.com/webmcu-ai) github organization.

If you download and unzip this repository you have access to all the working on-device and webmcu flash files already created.


WebMCU pages have web support for specific webserial firmware, which is very different from on-device firmware which is the bare bones on the xiao ml kit code and flashing files.


Each folder has the .bin binary to flash and the .ino so you can see the file loaded on the arduino IDE or platformIO
